도구로 돌아가기 개인정보 처리방침 문의

이용 안내

통합 이미지 유틸리티는 브라우저에서 실행되는 무료 이미지 작업 도구입니다.

주요 기능

PSD 레이어 추출, 스프라이트 시트 생성, 픽셀 이미지 확대, 파일명 교체 ZIP 생성을 지원합니다.

파일 처리

파일은 사용자의 브라우저에서 처리됩니다. 큰 PSD나 많은 이미지는 PC 성능에 따라 처리 시간이 길어질 수 있습니다.

결과 저장

웹 브라우저 보안 정책상 원본 파일을 직접 덮어쓰지 않습니다. 결과물은 PNG 또는 ZIP으로 다운로드한 뒤 사용자가 직접 적용해야 합니다.

책임 범위

중요한 원본 파일은 사용 전 백업하는 것을 권장합니다. 도구 사용 결과는 사용자가 직접 확인해야 합니다.
